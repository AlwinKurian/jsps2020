Javascript Programming Spring 2020

Welcome to JSPS2020. This is going to be your default repo for submitting homeworks. Each week you will be assigned a task that you will complete and merge to this repo in your own page.

Task 1

  • Fork this repo
  • Add a new page - use your conestoga id as the page name e.g. lnaqvi.html
  • Your page should have your name as the title and h1, you can also use css to style your page
  • Use document.write() to add a paragraph under your name introducing yourself - add information like your hometown, education background, experience, hobbies and interests.
  • Push the new page to your forked repo and create a pull request.
